# рҹҮ®рҹҮ© Tentang Taskflow

**Taskflow** adalah aplikasi manajemen tugas berbasis Kanban yang dirancang untuk individu maupun tim kecil. Dibangun dengan teknologi modern dan fokus pada kemudahan penggunaan, Taskflow membantu kamu tetap terorganisir, produktif, dan sadar akan progres pekerjaan setiap harinya.

---

## вңЁ Fitur Lengkap

### рҹ—ӮпёҸ Kanban Board вҖ” 4 Kolom Cerdas

| Kolom | Fungsi | Sorting |
|---|---|---|
| **To Do** | Daftar task yang perlu dikerjakan | Manual (drag & drop) |
| **In Progress** | Task yang sedang dikerjakan | Otomatis by due date terdekat |
| **Waiting / Blocked** | Task yang tertahan (nunggu vendor, approval, dll) | Otomatis by due date terdekat |
| **Done** | Task yang selesai | Manual |

- вң… **Drag & drop** antar kolom dengan animasi halus
- вң… **Wajib isi alasan** saat memindah task antar kolom (dengan preset alasan cepat)
- вң… **Visual berbeda** untuk card Waiting/Blocked (border amber + badge pulse)
- вң… **Scroll per kolom** вҖ” maksimal ~10 task terlihat, lebih dari itu scroll

### рҹ“қ Task Detail yang Komprehensif

Setiap task bisa diisi dengan:
- **Judul & deskripsi** lengkap
- **Priority** вҖ” High / Medium / Low (dengan multiplier poin)
- **Due date** вҖ” otomatis highlight merah jika overdue, kuning jika hari ini
- **Assignee** вҖ” nama atau email PIC
- **Labels** вҖ” preset + custom label
- **Status** вҖ” bisa diubah langsung dari modal

### рҹ“Ӣ Tahapan Task (Subtask)

Setiap task bisa punya tahapan/subtask sendiri:
- Tambah tahapan satu per satu dengan nama + PIC email
- Status per tahapan: **To Do вҶ’ In Progress вҶ’ Done** (klik untuk cycle)
- **Progress bar** langsung terlihat di card kanban
- List tahapan bisa di-expand/collapse langsung dari card
- Nama PIC singkat (username sebelum @) ditampilkan di card

### рҹ•җ Activity Log & Update

- **Activity log** per task вҖ” semua perubahan tercatat otomatis
- **Quick note** langsung dari card (textarea inline, вҢҳEnter untuk kirim)
- **Add Comment** panjang via tab di modal edit task
- Log termasuk alasan perpindahan kolom
- Tampilan note berbeda (bubble ungu) vs log sistem

### рҹ“… Kalender Aktivitas

- **View bulanan** вҖ” grid 7 kolom per minggu, persis GitHub contribution graph
- **View tahunan** вҖ” 12 bulan sekaligus, kotak mini per hari
- **3 layer data**: Due date (abu-abu) В· Dibuat (biru) В· Selesai (hijau)
- **Intensitas warna** вҖ” makin banyak task, makin pekat
- **Hover tooltip** вҖ” detail jumlah per kategori per hari
- Filter layer: Semua / Due Date / Dibuat / Selesai

### рҹ“Ҡ Laporan Performa

- **Trend 6 bulan** dengan grafik garis SVG + bar chart
- **4 KPI cards**: Selesai, Dibuat, Completion Rate%, Overdue
- **Badge tren** vs bulan lalu (вҶ‘+30% / вҶ“-10%)
- **Tabel detail** breakdown per bulan
- **Insight otomatis** вҖ” evaluasi berdasarkan data nyata
- Switch chart: Selesai / Dibuat / Rate% / Notes

### рҹҸҶ Productivity Score

Sistem poin gamifikasi dengan penambah dan pengurang:

**Penambah:**
| Aksi | Poin |
|---|---|
| Task selesai (Done) | +5 Г— priority |
| Pindah kolom | +2 Г— priority |
| Buat task baru | +1 Г— priority |
| Tambah note/update | +1 Г— priority |
| Priority multiplier | HighГ—3 В· MedГ—2 В· LowГ—1 |

**Pengurang:**
| Kondisi | Penalti |
|---|---|
| Hari tanpa aktivitas | вҲ’1 per hari |
| Task overdue | вҲ’2 per task |
| Minimum skor | 0 (tidak bisa minus) |

- **3 periode**: Today / Week / Month
- **Rank system**: Newbie вҶ’ Starter вҶ’ Active вҶ’ Pro вҶ’ Elite вҶ’ Legendary рҹ‘‘
- Stacked bar visualisasi gained vs penalty

### рҹ”” Email Reminder Otomatis

- **H-1 deadline** вҶ’ email pengingat 1 hari sebelum
- **Overdue** вҶ’ email dikirim setiap hari sampai task diselesaikan atau due date diupdate
- **Jadwal**: setiap hari jam 06:00 WIB (via pg_cron)
- Email HTML yang rapi dengan detail task, priority, dan status
- Powered by **Resend** + **Supabase Edge Functions**

### рҹ”җ Auth & Keamanan

- Email + password sign up / sign in
- Session persists (tidak perlu login ulang)
- **Row Level Security (RLS)** вҖ” setiap user hanya bisa akses data miliknya sendiri
- Supabase Auth terintegrasi langsung

### рҹ“ұ Mobile Friendly

- Kolom stack vertikal di mobile, jejer di desktop
- **Touch drag & drop** вҖ” hold 200ms lalu seret
- Navbar responsif вҖ” label teks disembunyikan di layar kecil
- Scroll per kolom tetap berfungsi di mobile

## рҹ—„пёҸ Database Schema

```
auth.users          вҶҗ Supabase built-in Auth

tasks               вҶҗ Task utama
  id, user_id, title, description, status, priority,
  due_date, labels[], assignee, position,
  created_at, updated_at
```

## рҹҡҖ Setup & Deploy

### Prerequisites
- Node.js 18+
- Akun [Supabase](https://supabase.com) (gratis)
- Akun [Vercel](https://vercel.com) (gratis)
- Akun [Resend](https://resend.com) (gratis, 3.000 email/bulan)

### 1. Clone & Install
```bash
git clone https://github.com/junn-a/kanban-project.git
cd kanban-project
npm install
```

### 2. Setup Supabase
1. Buat project baru di [supabase.com](https://supabase.com)
2. SQL Editor вҶ’ New Query вҶ’ paste & run `supabase-schema.sql`
3. SQL Editor вҶ’ run `supabase-migration-steps.sql`
4. Settings вҶ’ API вҶ’ copy **Project URL** dan **anon key**

### 3. Environment Variables
```bash
cp .env.example .env
```
Edit `.env`:
```env
VITE_SUPABASE_URL=https://xxxxxxxxxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGci...
```

### 4. Run Development
```bash
npm run dev
# вҶ’ http://localhost:5173
```

### 5. Deploy ke Vercel
```bash
# Via CLI
npm i -g vercel
vercel

# Atau push ke GitHub вҶ’ import di vercel.com
# Tambah env vars: VITE_SUPABASE_URL + VITE_SUPABASE_ANON_KEY
```

### 6. Setup Email Reminder (Opsional)
Lihat panduan lengkap di [`REMINDER_SETUP.md`](./REMINDER_SETUP.md)

# рҹҮ¬рҹҮ§ About Taskflow

**Taskflow** is a Kanban-based task management application designed for both individuals and small teams. Built with modern technologies and focused on ease of use, Taskflow helps you stay organized, productive, and aware of your work progress every day.

## вңЁ Key Features

- **4-Column Kanban Board** вҖ” To Do, In Progress, Waiting/Blocked, Done
- **Smart Sorting** вҖ” In Progress & Waiting auto-sorted by nearest due date
- **Task Steps** вҖ” Subtask/checklist with PIC email per step, progress bar on card
- **Move Reason** вҖ” Required reason when moving tasks between columns (with presets)
- **Activity Log** вҖ” Every change logged automatically, viewable per task
- **Quick Notes** вҖ” Inline notes directly from card or long comments via modal
- **Activity Calendar** вҖ” GitHub-style heatmap, monthly & yearly view, 3 data layers
- **Performance Report** вҖ” 6-month trend chart, KPI cards, auto insights
- **Productivity Score** вҖ” Gamified points with daily idle penalty & overdue penalty
- **Email Reminders** вҖ” Automated H-1 & daily overdue reminders via Resend + pg_cron
- **Mobile Friendly** вҖ” Responsive layout with touch drag & drop support
- **Secure by Default** вҖ” Row Level Security ensures users only see their own data

## рҹӣ пёҸ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + Tailwind CSS |
| Drag & Drop | @dnd-kit |
| Database | Supabase (PostgreSQL) |
| Auth | Supabase Auth |
| Realtime | Supabase Realtime |
| Email | Resend API |
| Cron | Supabase pg_cron + Edge Functions |
| Hosting | Vercel |
